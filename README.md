# 图片生成捷径字段

## 功能描述

该捷径字段用于生成和展示图片，支持单次生成多张图片。

## 实现说明

1. **图片生成**：调用阿里云DashScope API生成图片
2. **多张图片支持**：每次生成可以返回多张图片
3. **图片展示**：按照指定格式返回图片附件，支持直接显示

## 注意事项

- 每次调用生成的图片都是独立的，当前环境不支持持久化存储历史图片
- 如果需要保留历史图片，建议在多维表格中使用多个字段或记录来存储
- 或者考虑使用其他方式来管理历史图片，例如外部数据库或文件存储

## 使用说明

1. 配置图片URL和编辑指令
2. 运行捷径生成图片
3. 查看生成的图片结果

## 技术栈

- Node.js
- TypeScript
- @lark-opdev/block-basekit-server-api
